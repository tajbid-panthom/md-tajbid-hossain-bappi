# TajDEV Portfolio

Personal portfolio website for **Md Tajbid Hossain Bappi**, built with Next.js, Tailwind CSS, and shadcn/ui.

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **shadcn/ui** components

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — Home (hero, skills preview, projects, testimonials)
- `/about` — Skills with animated progress bars
- `/portfolio` — Detailed project showcase
- `/services` — What I do (no pricing)
- `/contact` — Contact form and info

## Legacy

The original HTML/CSS portfolio is preserved in the `legacy/` folder.
